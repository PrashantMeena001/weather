# 🌦️ Weather App

This is a simple weather application that I built using React and Vite.
It allows users to search for any city and get real-time weather information like temperature, humidity, and wind speed.

The goal of this project was to understand how APIs work and how to manage data flow in a React application.

---

## What it does

* Search weather by city name
* Shows current temperature
* Displays humidity and wind speed
* Updates data in real-time using an API
* Clean and simple user interface

---

##  Tech used

* React (with Vite)
* Axios for API calls
* Visual Crossing Weather API (via RapidAPI)
* Basic CSS for styling

---

##  How to run this project

1. Clone the repo

```bash
git clone https://github.com/PrashantMeena001/weather.git
```

2. Go into the folder

```bash
cd weather
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env` file and add your API key

```bash
VITE_API_KEY=your_api_key_here
```

5. Start the project

```bash
npm run dev
```

6. Open in browser

```
http://localhost:5173
```

---

## Note

You will need your own API key from RapidAPI for this project to work.
The `.env` file is not included in the repo for security reasons.
